# 后台管理系统

接口文档：https://www.showdoc.com.cn/128719739414963?page_id=2516997897914014

该项目为一个微信小程序商城前端，项目使用小程序原生框架开发，包含商城首页、商品分类页、个人中心、购物车等页面，实现了一个功能基本完整的商城小程序
